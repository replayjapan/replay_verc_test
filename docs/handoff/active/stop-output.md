## 5. Design Learnings

### What Worked
- Vertical stacking with flexDirection column override for Payload card layout
- Large headline numbers (36px count, 28px score) for quick scanning
- Green/amber/red freshness indicators for at-a-glance status
- Japanese 万/億 shorthand for portfolio value

### What Didn't Work
- Horizontal grid layouts cramped in Payload's narrow card containers — required 3 iterations
- Missing defaultLayout caused widgets to not appear at all on first login

### Patterns to Carry Forward
- Always add `display: flex; flexDirection: column` to Payload dashboard widget wrappers
- Use `defaultLayout` to set initial positions
- Keep `beforeDashboard` for interactive admin actions (client components)
