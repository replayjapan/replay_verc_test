# Framework Feedback Log

Track inconsistencies, gaps, and improvement ideas discovered during real usage.

## How This Works
- **PM (ChatGPT/Claude/Gemini):** Outputs a FRAMEWORK FEEDBACK section at the top of every plan. Developer copies notable items here.
- **Claude Code:** Optionally appends entries when it encounters missing guidance, contradictions, or gaps during execution. At the end of a feature, Claude Code mentions if feedback was added.
- **Developer:** Reviews periodically and decides what to fix in the framework.
