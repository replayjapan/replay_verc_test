# Plan Templates

Reusable templates for milestone planning and execution.
Source of truth lives in PL_Agent/framework/templates/ — these are project copies.

## Files

| Template | Used By | When |
|----------|---------|------|
| `STOP_TEMPLATE.md` | EngAI | Every STOP gate |
| `PLAI_REVIEW_TEMPLATE.md` | PLAI | After reviewing each STOP |
| `KICKOFF_TEMPLATE.md` | PMAI | Writing each milestone kickoff |
| `PMAI_DIRECTION_TEMPLATE.md` | PMAI | Mid-milestone plan adjustments |
| `HANDOFF_TEMPLATE.md` | EngAI | At milestone ship |
| `PLAN_TEMPLATE.md` | PMAI | Writing each milestone plan |
| `plan_state.template.json` | PMAI | Creating plan_state.json for each milestone |
