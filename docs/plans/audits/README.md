# Audit Reports

Design, security, and other audit reports organized by milestone.

## Structure

```
audits/
├── m22-design-audit/
│   ├── engai-audit.md
│   ├── codex-audit.md
│   └── design-direction.md
└── m23-security-audit/
    └── security-report.md
```

## Rules

- Milestone number and audit type in folder name
- Multi-AI audits: EngAI and Codex write separate docs, never cross-contaminated
- PLAI reviews both, Developer decides
