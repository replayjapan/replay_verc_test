# Content Briefs

Prompts and instructions for content generation tasks, organized by milestone.

## Structure

```
briefs/
├── m22a-codex-domain-content/
│   └── codex-production-prompt.md
├── m22b-codex-blog-content/
│   └── codex-blog-prompt.md
└── m25-codex-styleguide/
    └── codex-styleguide-prompt.md
```

## Rules

- PLAI writes the brief, Developer approves before sending to Codex/Cowork
- Brief must reference which governance files to read (voice-brief, rubric, etc.)
- Brief must specify output format (JSON for seeds, markdown for docs)
- Milestone number and name in folder name
